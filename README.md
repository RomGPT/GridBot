# GridBot

## DIP-HUNTER BOT V3.0

Репозиторий содержит модуль `src/dip_hunter_bot.py`, реализующий архитектуру
бота Dip Hunter версии 3.0 для торговли резких падений. Реализация следует
описанной стратегии Multi-Layer Dip Buying и включает:

* систему многоуровневой фильтрации сигналов;
* расчёт размера позиций в зависимости от силы падения;
* управление входами, рисками, комиссионными издержками;
* отчётность по производительности и адаптивную оптимизацию параметров;
* чек-лист перед деплоем и поэтапное масштабирование.

Для демонстрации предусмотрены мок-реализации провайдера маркет-данных и
исполнения ордеров. Эти классы можно заменить на реальные интеграции с Bybit
или другими брокерами.
