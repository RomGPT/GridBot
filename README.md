# GridBot

GridBot is a small trading-bot project workspace.

## Status

The repository is active and ready for the next implementation steps. Current focus areas:

- define the trading strategy inputs and risk limits;
- add configuration for exchange/API credentials outside the codebase;
- prepare a simple backtesting loop before live trading;
- document setup and run commands as the bot grows.

## Notes

Keep secrets out of the repository. Use environment variables or a local config file that is ignored by Git.
